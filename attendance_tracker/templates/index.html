Club Attendance Tracker Project

Welcome to app home page! This project was created for the purpose of increasing visibility of WSU club room usage. Originally started as a student project for the CptS 322 Software Engineering course, the project aims to give club members and club admin basic tools to assist with making data informed decisions regarding club resources. Data available through the analytics pages are sourced from WSU's IT group and include anonymized building access data. The app covers several functions accessible through the navigation bar in the top right corner of the screen. Brief descriptions of each app section will be described below!

Login page. As expected, the login page allows admin users to log into the system in order to access privileged pages containing data configuration features. Unauthenticated users attempting to access protected admin pages will be redirected back to the home page! Full description of admin pages will follow.

Admin pages. As mentioned previously, admin pages allow authenticated users to add and modify existing club information. These resources are available through several pages:

  • Add Club page: record the creation of a new club.
  • Assign Room page: link an existing club to a room.
  • View Clubs: View all registered clubs in a table format, follow an edit link modify any entry.

Analytics pages. The analytics pages offer two simple views for looking at room usage over time. Both views are based on room accesses by date.

  • Activity Trend page: see weekly access trends for a space over a selected date range.
  • Room Usage page: See cumulative accesses for a set of spaces over a selected date range.
